Always on Top
==============
Set the website to Always-on-Top (aka Picture-in-Picture, CompactOverlay, Mini View).  
https://www.microsoft.com/store/apps/9p37ft08rnxm

## Feature
Share target to receive the website URL

## Test
Tested on Windows 10 Version 1803
